# ToDo List

## _Lista de Tarefas_

O projeto ToDo List é um pequeno projeto no qual pode-se adicionar tarefas pendentes e removê-las, facilitando o gerenciamento de tempo.

## Tecnologias:
As tecnologias utilizadas foram:

- HTML
- CSS
- JavaScript
